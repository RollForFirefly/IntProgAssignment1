## Programming on the Internet Assignment 1 - Shopping Cart Webpage

This repository contains the frontend and backend of a locally-hosted webpage that provides a shop system. It connects to a database created with MySQL and SQLModel. It uses Node.js, Javascript, and Python.

The project is separated into folders for the frontend and the backend of the system. Frontend includes all React, Javascript, and CSS files. Backend includes python files that connect to the database.

This project was an interesting challenge as I have little experience with local webservers prior to this subject. Most of my coding experience has been in game engines.
